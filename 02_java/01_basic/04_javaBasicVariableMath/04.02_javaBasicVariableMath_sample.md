# 📘 04.02_javaBasicVariableMath_sample.md

## 🧠 Math
* common math operators to perform calculations.

#### Java Arithmetic Operators


##### ✏️ Example: `+` addition math operators

```java
int a = 5;
int b = 10;
int c = a + b;
System.out.println(c + b);
```

##### 🔥 output

```
25
```

----

##### ✏️ Example: `-` subtraction math operators (Age)
  
```java
int myAge = 16;
int myFriendAge = 14;
System.out.println(myAge - myFriendAge);
```

##### 🔥 output

```
2
```

----

##### ✏️ Example: `-` subtraction math operators (Tax)

```java
int monthlySalary = 10000;
int monthlyTax = 1500;
int monthlySalarySfterTax = monthlySalary - monthlyTax;
System.out.println(monthlySalarySfterTax);
```

##### 🔥 output

```
8500
```

----

##### ✏️ Example: `*` Multiplication math operators (Discount)


##### example #1: print price after discount.

```java
int price = 100;
System.out.println(price * 0.50);
```

##### 🔥 output

```
50
```

----

##### example #2: print price after discount.

```java
double discountPercent = 0.50;
double price = 100;
double priceAfterDiscount = price * discountPercent;
System.out.println(priceAfterDiscount);
```

##### 🔥 output

```
50
```

----

##### example #3: print building Height (total Floors * floor Height) .

```java
int totalFloors = 3;
int floorHeight = 3;
int buildingHeight = totalFloors * floorHeight;
System.out.println(buildingHeight);
```

##### 🔥 output

```
9
```

----

##### ✏️ Example: `/` devision math operators (how many candies can i buy).

```java
int myMoney = 120;
int oneCandy = 6;
int boughtCandy = myMoney / oneCandy;
System.out.println(boughtCandy);
```

##### 🔥 output

```
20
```

----

##### ✏️ Example: `%` modulo (remainder) math operators (get money change)

```java
		int money = 13;
		int candy = 5;
		int changeLeft = money % candy;
		System.out.println(changeLeft);
```

##### 🔥 output

```
3
```

### 📌 Notes

* Basic math operators to perform calculations with values and variables.
+ is addition
- is subtraction
/ is division
% finds the modulo/remainder of a division.

You can add Strings using the + operator, in a process called concatenation.
