## 🧠 Math
* common math Arithmetic operators to perform calculations (+ - * / %).

#### Java Arithmetic Operators

| symbol | name | example |
| :-: | :-: | :-: |
| + | addition | a + b | 
| - | subtraction | a - b | 
| * | multiplication | b * a | 
| / | devision | b / a | 
| % | modulo | b % a | 

----- 

##### ✏️ Example: `+` addition math operators

```java
int myAge = 16;
int myFriendAge = 14;
System.out.println(myAge - myFriendAge);
```

##### 🔥 output

```
2
```
