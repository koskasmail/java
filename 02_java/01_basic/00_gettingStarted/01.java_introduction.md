### ‚òï what is java ?

* Java is Platform Independent, which means that you only need to write the program once 
   to be able to run it on a number of different platforms (operating system: windows/linux/ubuntu).
* Java is used to: 
    * develop apps for Google's Android OS
    * various Desktop Applications 

### ‚òï Key Features of Java
* `Object-Oriented`:
   * Java is built around the concept of objects and classes, promoting modular and reusable code.
* `bytecode` Platform Independent:
   * Java code is compiled into `bytecode` that runs on the Java Virtual Machine (JVM),
      * allowing it to run on any device with a JVM
   * the famous "write once, run anywhere" principle.
* `Strongly Typed and Secure`:
   * Java enforces strict type checking and includes
      * built-in security features like runtime checking and garbage collection.
* `Multithreaded and Concurrent`:
   * Java supports multithreading,
      * enabling programs to perform multiple tasks simultaneously for better performance.
* `Rich Standard Library`:
   * Java comes with a vast set of libraries and APIs for
      * networking, data structures, GUI development, and more.

### üõ†Ô∏è Common Uses of Java
* `Android App Development`:
   * Java is one of the primary languages for Android apps.
* `Web Applications`:
   * Java powers many enterprise level web applications using frameworks like
      * Spring and Java_EE.
* `Desktop Applications`:
   * Tools like Eclipse and IntelliJ IDEA are themselves built in Java.
* `Big Data and Cloud Computing`:
   * Java is used in platforms like
      * Hadoop and Apache Spark.
* `Games and Embedded Systems`:
   * Java is also used in
      * game development and embedded device software.
