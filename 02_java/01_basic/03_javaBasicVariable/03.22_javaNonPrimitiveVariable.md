# 📘 03.22.javaNonPrimitiveVariable

## 🧠 Variable Non-Primitive Data Types

* Non-Primitive Data Types are: String, Array, Interface, Classes
* variable type are written in lowercase
* variable Names must begin with a letter <a..z>

----

## 🧬 Non-Primitive Data Types

* Non-primitive types are more complex and include Array, objects and collections.
    * Examples: `String`, `Array`, `Class`, `Interface`
* Non-primitive types are **not predefined** like primitives.
* They can store **multiple values**, behaviors, or structures.


### 📝 variable Naming Conventions for Non-primitive types 
* variable Names can contain `letters`, `digits`, `underscores`, `dollar signs <$>`
* java naming convention for a `Variables` name format: begin with a `letter` (`a`–`z`) and start with a `lowercase` letter.
* Variable names can contain `letters`, `digits`, `underscores`, `dollar signs` (`$`).


### 🧵 Example

```java
String userName = "Jaron";
int[] scores = {90, 85, 88};
```

----

### 📌 Notes

- Java is **case-sensitive**: `myVar` and `MyVar` are different.
- Use **meaningful names** to improve code readability.
- Avoid starting variable names with `$` unless necessary (e.g., generated code).
