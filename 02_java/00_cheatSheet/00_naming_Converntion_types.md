# 00_naming_Converntion_types

| Convention | Few Words |  Example |
|-----------|-----------|----------|
| PascalCase | Capital Each Word |  UserService, MyOsMain |
| camelCase | first word lower + Next Words Capital | loadUsers, variables |
| UPPERCASE  | ALL CAPS uppercase | MAX_USERS, SERVER_PORT |
| lowercase | all small letters | package myos, import handlers |
| snake_case | lower case words with uderscores between words | user_id, created_at |
| Capital Letter | first letter uppercase | ClassName, UserInfo MyId| |
