# 03_java_naming_convention

# 📘 Quick Java Summary Table

| Convention | Few Words | Used For | Example |
|-----------|-----------|----------|---------|
| PascalCase | Capital Each Word | Classes, Interface, Enums, Records| `UserHandler` |
| camelCase | lower + Capital | methods, variables | `loadUsers()` |
| UPPERCASE | ALL CAPS / all UPERCASE | Constants | `MAX_USERS` |
| lowercase | all lower case package | package, project | `bank`, `credit card` | 
| snake_case | lower_case_words | JSON, config | `user_id` |
